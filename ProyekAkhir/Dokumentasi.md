# Kriteria Penilaikan
## Proposal Proyek (5%)

- Kejelasan ide dan tujuan proyek
- kesesuaian dengan materi perkuliahan
- kelayakkan proyek dengan batasan waktu yang diberikan

## Pengembangan Frontend (20%)

- penggunaan javascript modern (ES6+, seperti arrow function, promises, await/async)
- Desain yang responsif dan pengalaman pengguna yang baik
- Integrasi dengan API Backend

## Pengembangan Backend (25%)

- penggunaan Express.js yang tepat untuk membangun API RESTfull
- implementasi Autentikasi dan otorisasi yang aman (OAuth2)
- operasi CRUD dengan desain skema database yang sesuai

## Keamanan dan Pengujian (15%)

- implementasi langkah-langkah keamanan web (seperti validasi input, HTTPS)
- pengujian API (menggunakan postman, unit test, dll)
- Penanganan error dan pengecualian yang baik

## Dployment (15%)

- berhasil menerapkan frontend dan backend
- penggunaan ipeline CI/CD untu otomatisasi deployment
- Aplikasi yang dapat diakses pengguna

## Skalabilitas dan Optimasi (10%)

- penggunaan teknik peningkatan performa (seperti catching, query yang efisien)
- pertimbangan untuk menangani peningkatan jumlah pengguna

## Presentasi dan Dokumentasi (10%)

- kejelasan dan profesionalitas saat presentasi
- dokumentasi lengkap (Readme, spesifikasi API, langkah deployment)

## Kerja Tim dan Kontribusi (10%)

- distribusi tugas yang merata antara anggota tim
- kolaborasi dan pemecahan masalah sebagai sebuah kelompok
